圆周运动 Circular Motion

🎯 学习目标

理解匀速圆周运动的向心力与向心加速度概念
掌握向心力公式 $F = mv^2/r$ 和 $a = v^2/r$
理解角速度 $\omega$、半径 $r$ 对向心力的影响
学会计算圆周运动周期 $T = 2\pi r / v$

📐 核心公式

$a = \frac{v^2}{r}$
$F = m\frac{v^2}{r}$
$T = \frac{2\pi r}{v}$
$v = \omega r$

💡 原理解读

圆周运动是自然界中最基本的运动形式之一。

🎢 想象你坐在旋转木马上——
🔗 虽然速度大小不变，但方向始终在改变
💥 这种方向改变需要 向心力 持续作用

关键理解：匀速圆周运动中，速度大小恒定，但方向不断变化——这本身就是一种加速度（向心加速度）。

⚠ 红色箭头 = 向心力，蓝色箭头 = 线速度方向
