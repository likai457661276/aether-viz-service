溶质、溶剂、溶解过程

学习目标

核心概念

溶液 = 溶质 + 溶剂

NaCl + H₂O → NaCl溶液

溶解度公式

S = (m溶质 / m溶剂) × 100，单位：g

影响因素

① 温度（大多数固体随温度升高溶解度增大）
② 压强（气体溶解度随压强增大而增大）
③ 溶剂性质
