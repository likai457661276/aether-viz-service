整式加减

初一数学 · 第六章

核心法则
运算步骤

步骤一：去括号
括号前是"+"，去括号后不变号
括号前是"−"，去括号后要变号

步骤二：合并同类项
只把同类项的系数相加减
字母和字母的指数不变

真实应用
📐 周长计算：2(a+b) + 2a
📊 数据整理：求和后合并
💰 价格计算：(a+b) + (a−c)

🧮 整式加减计算

3x²+2x−−x²−−5x
(3x² + 2x) − (−x² − 5x)
原式： (3x² + 2x) − (−x² − 5x)
去括号： 3x² + 2x + x² + 5x
交换位置： 3x² + x² + 2x + 5x
合并同类项： (3+1)x² + (2+5)x
✓ 结果： 4x² + 7x
