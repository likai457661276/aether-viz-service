二次函数 Quadratic Function

高一数学 · 抛物线、顶点、对称轴与零点的动态实验

学习目标

核心公式

y = ax² + bx + c
xᵥ = -b / 2a，yᵥ = f(xᵥ)
Δ = b² - 4ac

当 a > 0 时，抛物线向上开口，顶点是最低点；当 a < 0 时，抛物线向下开口，顶点是最高点。b 主要改变对称轴位置，c 是图像与 y 轴的交点。

原理解释

二次函数像一条可以弯曲、翻转、平移的轨道。参数 a 控制轨道的“弯曲力度”和开口方向，b 会把对称轴推向左右，c 决定轨道穿过 y 轴的高度。拖动滑块时，图像不是在背公式，而是在把代数信息直接变成几何形状。

为什么重要

抛物线出现在抛体运动、桥拱设计、利润最大化、信号拟合和机器学习损失函数中。掌握顶点和零点，就是学会从变化中找关键位置。
